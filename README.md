# mkright-archive

## 项目简介

`mkright-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名、网站或在线服务，仅作为静态 HTML 内容的整理与存储空间，供浏览、查阅或历史记录参考。

## 目录说明

```
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── index.html      # 仓库首页或导航页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件均为一个独立的归档页面，可单独打开浏览。
- `assets/` 目录用于存放页面所需的共用或独立资源，按需组织。

## 页面归档说明

- 每个 HTML 页面均为完整、自包含的文档，部分页面可能引用 `assets/` 下的资源以保持功能或样式。
- 页面内容可能包含不同时期、不同来源的信息，均以归档形式呈现，不代表当前状态或最新版本。
- 归档页面可能保留原始排版与结构，不进行内容修改或美化。

## 维护说明

- 本仓库由维护者不定期更新，新增或替换页面时尽量保持目录结构清晰。
- 不保证页面内容的完整性、准确性或时效性。
- 欢迎通过 Issues 提出关于归档内容的问题或建议，但不对具体页面内容承担更新义务。
- 本仓库仅用于静态归档目的，不提供任何形式的交互服务或外部链接。

## 使用方式

1. 直接通过 GitHub Pages 或本地文件系统打开 `index.html` 或 `pages/` 下的任一 HTML 文件。
2. 克隆仓库后，在浏览器中打开本地 HTML 文件即可查看。

## 许可

本仓库内容仅供个人归档与学习参考使用，不涉及商业用途。具体页面可能包含其原始来源的版权，请尊重原作者权益。
